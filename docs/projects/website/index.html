The Big Idea

There’s nothing particularly exceptional about this site, I’ll admit. It’s HTML and CSS. What else does it need? Not a thing. Every developer should have a site, I think. It’s just a few simple pages so you can get to know me a little better.

Stack:

  • Node
  • Express
  • 11ty/Nunjucks
  • Tailwind
  • Git Repo

Development

Cutting Edge

For the 6th version of my personal site, I’ve transitioned to using 11ty, a static site generator. I serve the build on an EvenNode server with Express.

Strategy

Not a Blog

I previously had a blog API serving every article I’ve ever written about the joys of designing and developing. But looking back at 2013 Me discovering simple things got a little depressing.

Creativity

Ain’t It Pretty

Again, there’s nothing particularly exceptional here. I really needed to spend more time developing than designing this site. Every project has it’s priorities and over-designing this simply wasn’t it.
