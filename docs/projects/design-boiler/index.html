Design Boiler

A Boilerplate for Rapid Prototyping

The Big Idea

I built the same basic components in two (planned for three) different versions of frameworks. One source set of styles applies to all of the different versions of HTML. Theoretically, these can be used as easily tweak-able examples while rapid prototyping, no matter which framework you choose for your project.

Stack:

  • HTML 5
  • SCSS
  • Dreamweaver templates
  • Sample Site
  • Git Repo

Development

A Head Start

While it may be a little more difficult to jump in and out of frameworks and keep them all straight, choosing a framework and going with it for an entire project makes it 100% easier for developers to jump in and code without too much ramp-up time.

Creativity

Cutting Corners

As a person who’s designed a web thing or two, I very much appreciate the design coverage. I appreciate being able to change as little or as much as I want to customize my design. When prototyping, this is essential.
