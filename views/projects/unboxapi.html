Blog API

This API is in progress, but - for now - it feeds my blog post archive from a mySQL database to this website. It has no frills or extra features, just displays the posts. I'm planning to add blog "likes" and comments back as well as an administration center. I'll get there, just slowly is all. Consider it the least I could do.

RSS Feed

Along with the API, it feeds the archive into an RSS feed. Again, nothing fancy. Just a feed of text-based posts.
