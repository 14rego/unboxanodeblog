I’ve been hearing quite a lot about Foundation (a front-end framework) lately. I decided to look into it and see what it’s all about.

As a long-time [in developer years] Twitter Bootstrap user, I’ve got to say — I’m slightly blown away. Now, I haven’t tried it yet, but I can already see a lot of grid features I wish Bootstrap had. I found enough features I felt like I couldn’t live without that I’m convinced I’ll be trying out Foundation for a small project very soon! Stay tuned!
