[ Another from the KCDC series… ] I recently went to the Kansas City Developer’s Conference where I like to think I learned a lot and I’d like to share some with you.

Topics discussed included the Top 10 Types of Security Vulnerabilities as put forth by the Open Web Application Security Project.

What I found most interesting:

Broken Authorization and Session Management — Scripts can be used to compromise credentials, steal sessions, and assume an unknown identity. Always generate session IDs randomly, pass only the session ID to the browser, change the ID regularly, store the IP and ID and check them regularly!
