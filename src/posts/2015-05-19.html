I recently found out about CSS Viewport Units. It would be fantastic to use these units to make things like an easy 960 grid.

I feel like I’d heard of it before, but was discouraged by the browser support. Looking at it again, I’m a little more optimistic, but those are some crazy “known issues”! “iOS 7 Safari sets viewport unit values to 0 if the page has been left and is returned to after 60 seconds.” “iOS 7 Safari recalculates widths set in vh as vw, and heights set in vw as vh, when orientation changes.” After looking at the support list, I was *most* excited to use this in iOS. Not anymore.
