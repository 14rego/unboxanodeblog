I was recently working on a web site that I needed to use pseudo-selectors in jQuery. I don’t usually use a lot of pseudo-selectors because I almost always have to allow full IE8 compatibility. But it suddenly occurred to me to ask the question, “Just because I can’t use pseudo-selectors in my CSS, does that necessarily mean that I can’t use them at all?”

The answer is no. As this jQuery Browser Support page outlines, as long as you’re not using the .css() method to change your CSS, you can use all the pseudo-selectors in your $('.selectors:nth-child(0)') as you wish! I learned something today. I hope you did, too.
